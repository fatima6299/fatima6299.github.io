# Portfolio Personnel - Fatimata Tidiane Diaz

Bienvenue sur mon portfolio personnel ! Ce site présente mon parcours professionnel, mes compétences et mes projets dans le domaine des télécommunications et du développement.

## Fonctionnalités

- **Design moderne et réactif** : S'adapte à tous les appareils (ordinateurs, tablettes, téléphones)
- **Mode sombre/clair** : Basculement automatique en fonction des préférences du système
- **Animations fluides** : Défilement doux et animations au défilement
- **Formulaire de contact** : Pour me contacter facilement
- **Portfolio interactif** : Mise en avant des projets avec des cartes cliquables
- **Navigation fluide** : Menu de navigation fixe et bouton de retour en haut de page

## Technologies utilisées

- HTML5 sémantique
- CSS3 avec variables CSS et animations
- JavaScript natif (ES6+)
- Font Awesome pour les icônes
- Google Fonts (Poppins et Inter)

## Comment l'utiliser

1. Clonez ce dépôt
2. Ouvrez le fichier `index.html` dans votre navigateur
3. Explorez les différentes sections du portfolio

## Personnalisation

Pour personnaliser ce portfolio :

1. Remplacez les images dans le dossier `assets/img/`
2. Mettez à jour les informations personnelles dans `index.html`
3. Modifiez les couleurs dans `style.css` (variables CSS dans `:root`)
4. Ajoutez vos propres projets dans la section dédiée
